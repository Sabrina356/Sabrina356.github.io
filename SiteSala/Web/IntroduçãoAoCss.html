Introdução ao Css



  O Css(Folha de estilo em cascata) assim como o html não é uma linguagem de programação,ele é uma linguagem de estilização utilizado para estilizar um documento HTML,ou seja para tornar o documento mais dinamico. O HTLM controla a estrutura da pagina e o Css determina sua aparencia.


Aplicando o Css

Para aplicar o Css dentro do HTML exitem regras de formatação no qual essas regras podem ser aplicadas de três formas distintas:

  • Inline: Que são estilos aplicados diretamente em uma elemento em HTML, dentro do documento
  • Incorporado: O elemento style que permite aplicar varias regras de uma vez no proprio documento HTML.
  • Folha de estilos externa: É um documento com extensão .css permite aplicar regaras a um website inteiro de uma vez.

Para aplicar a sintaxe de uma regra Css tem um formato que precisamos seguir. Sendo:

seletor{
 propriedade:valor;
}


 Onde o seletor será o alvo da regra e a declaração onde será aplicada os parametros de estilo que é composta por propriedade e valor.

Tipos de seletores

  1. Seletor Universal:Reprensenta todos os elementos contidos no textos simbolizado pelo simbolo *.
  2. Seletor de tipo:Representa um determinado elemento.
  3. Seletor de Classe:Aplica esilo a elementos com o Atribudo class definido. Classe essa que voce cria no documento HTML e para chama-la no css é :
    .nome da clasee{}.
  4. Seletor de ID:Aplica estilo a elementos com o atribudo ID definido. O ID voce tambem cria no HTML sendo que para chama-lo no css é :
    #nome do ID{}
